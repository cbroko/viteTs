Lurer du på hvilke egenskaper en pingvin kan ha?

Da har du kommet til ritkig kurs!

Hvis du ser på interface Penguin, så ser du alle egenskaper hvilken som helst pingvin kan ha. Men du skulle kanskje ønske at det fantes en union type for akkurat det?

Vi også! Har du lyst til å hjelpe oss å lage en union type basert på alle egenskapene til interface Penguin? Vi pleier å kalle denne typen for penguinKey...

Oppgave 8.3

Bouvetpingviner er ganske like vanlige pingviner, men det er noen åpenbare forskjeller. Derfor har noen laget type BouvetPenguin. Men det finnes en mer kompakt måte å gjøre dette på.

Se om du kan lage en mapped type av type CleverBouvetPenguin, for å oppnå det samme som type BouvetPenguin men med færre linjer.

Oppgave 8.4

Det finnes også flyktige pingviner, som ikke nødvendigvis har f.eks. et navn. Derfor har noen laget type VolotilePenguin. Men det finnes også en enklere måte å gjøre dette på.

Se om du kan lage en mapped type av type VolotileButCleverPenguin, for å oppnå det samme som type VolotilePenguin men med færre linjer.
